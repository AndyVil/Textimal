# Textimal (Text Minimal xD)
## A CodeMirror Implementation

Online Notebloc - Minimalist and private text editor with syntax highlighting. TXT, MD, Python, JS, TS, Rust, SQL, BASH, C, YML, XML, HTML, CSS

### Features:
Line Numbers \
Local storage \
Open Files \
Save File \
Copy Value\
Multi Lang \
Multicursor \
Word Wrap \
Light - Dark Theme \
Rename File \
Color Brackets \
Resposive \
PWA \
Offline \
Native SpellCheck (Bug - Disabled)

Next feature: Encoding

> You can download the file and run it locally, but an internet connection is required for first time because it uses CDN libraries.
